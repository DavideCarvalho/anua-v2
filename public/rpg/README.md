# RPG Assets

## Cute Fantasy (Kenmi Art, premium)

Chão e objetos em `cute-fantasy/` e `objects/`:
- **grass** (id 1) → Grass_Tiles_1–4.png (16 tiles variados)
- **water** (id 2) → Water_Middle.png
- **Mercadinho** → objects/market.png (House_1_Wood_Base_Red)

## 0x72 Dungeon Tileset II (CC0)

Em `0x72-dungeon/` — personagem (lizard_m) e objetos (chest).

- **Personagem**: `frames/lizard_m_idle_anim_f*.png`, `lizard_m_run_anim_f*.png`
- **Baú (treasure)**: `frames/chest_full_open_anim_f1.png`
- **Mailbox**: sprite programático

Para trocar o personagem, edite `external-player.ts` (ex.: `elf_m`, `dwarf_m`).
